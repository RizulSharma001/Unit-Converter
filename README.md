# ⚖️ Unit.Convert

A sleek, modern, and responsive unit converter designed with a focus on typography and user experience. It supports conversions for Length, Weight, and Temperature with a high-precision calculation engine.

## 🚀 Live Demo
You can use the application here:  
**[https://rizulsharma001.github.io/Unit-Converter/](https://rizulsharma001.github.io/Unit-Converter/)**

## ✨ Features
* **Three Modes:** Convert between Length (m, km, mi, etc.), Weight (kg, lb, oz, etc.), and Temperature (°C, °F, K).
* **Swap Functionality:** Instantly switch between "From" and "To" units with one click.
* **High Precision:** Handles large numbers and small decimals with scientific notation support.
* **Copy to Clipboard:** Copy your result with a single click.
* **Dark UI:** Minimalist aesthetic using the Syne and DM Mono typefaces.
